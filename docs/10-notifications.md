# 10 — Notifications

## Route

Dropdown panel in navbar + `/account/notifications` full page

## Notification types

- New message
- Someone favorited you
- Profile visitor ("45 people viewed you")
- Private album request / granted
- Verification approved
- Subscription renewal / expiry warning

## Features

- Dropdown with recent notifications
- Unread badge on bell icon
- Mark individual as read
- Mark all as read
- Realtime delivery via Supabase

## Gamification copy

Use curiosity-driving language:
- "23 members saved you"
- "Someone viewed your profile"

## Components

- `NotificationDropdown`, `NotificationItem`, `NotificationPage`
